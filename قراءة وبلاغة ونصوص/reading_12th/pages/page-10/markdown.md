# معاني المفردات والتراكيب اللغوية

|  الكلمة | معناها  |
| --- | --- |
|  الأنعام | هي الإبل والبقر والغنم.  |
|  فرث | الفرث ما يتبقى من فضلات الطعام في الكرش بعد الهضم.  |
|  سائغاً للشاربين | طيباً سهل الانحدار في الحلق، لا يغص به شاربه.  |
|  سكراً | السكر: الخمر، وقد نزلت هذه الآيات قبل تحريمها.  |
|  أوحى | ألهم وهدى.  |
|  مما يعرشون | مما يرفعون من الكروم وغيرها.  |
|  سبل ربك | فضاء الله الرحب من البراري والأودية والجبال وغيرها.  |
|  ذللا | مذللة مسهلة.  |
|  أرذل العمر | آخره في حال الكبر والعجز والحرف.  |
|  حفدة | الحفدة: أولاد البنين.  |
|  لا تضربوا لله الأمثال | لا تجعلوا لله أنداداً أو أشباهاً وأمثالاً.  |
