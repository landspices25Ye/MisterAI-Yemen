الأعداد المركبة

ثانياً : مقياس وسعة خارج قسمة عددين مركبين :

$‫\frac{ع_1}{ع_2} = \frac{س_1 (جتا هـ + ت جا هـ)}{س_2 (جتا هـ + ت جا هـ)} \quad [ \text{بضرب البسط والمقام في مرافق المقام} ]‬$‫

‬$‫= \frac{س_1 (جتا هـ + ت جا هـ) (جتا هـ - ت جا هـ)}{س_2 (جتا هـ + ت جا هـ) (جتا هـ - ت جا هـ)}‬$‫

‬$‫= \frac{س_1 (جتا هـ + جتا هـ - ت جتا هـ جا هـ + ت جا هـ جتا هـ - ت جتا هـ جا هـ)}{س_2 (جتا هـ + جتا هـ)}‬$‫

‬$‫= \frac{س_1 (جتا هـ + جتا هـ + جا هـ جا هـ) + ت (جا هـ + جتا هـ - جتا هـ جا هـ)}{س_2 (جتا هـ + جتا هـ)}‬$‫

بما أن : جتا هـ + جا هـ = ١ .

جتا ( هـ - هـ ) = جتا هـ + جا هـ + جا هـ

جا ( هـ - هـ ) = جا هـ + جتا هـ - جا هـ + جتا هـ

‬$‫\therefore \frac{ع_1}{ع_2} = \frac{س_1}{س_2} \quad [ \text{جتا ( هـ - هـ) + ت جا ( هـ - هـ) } = \frac{س_1}{س_2} \text{ ، هـ - هـ} ]‬$‫

‬$‫\text{مقياس } \left( \frac{ع_1}{ع_2} \right) = \left| \frac{ع_1}{ع_2} \right| = \frac{س_1}{س_2} = \left| \frac{ع_1}{ع_2} \right| \text{ ، ع ٢}‬$‫

‬$‫\text{سعة } \left( \frac{ع_1}{ع_2} \right) = هـ - هـ .‬$‫

مقياس خارج قسمة عددين مركبين يساوي خارج قسمة مقياسيهما وسعة خارج القسمة تساوي الفرق بين سعتيهما .

ثالثاً : حاصل ضرب عدد مركب ع في مرافقه يساوي مربع مقياس هذا العدد .

أي أن : ع = س ٢ + ص ٢ = |ع| ٢ .

رابعاً : مقلوب العدد المركب ع :

‬$‫\frac{1}{ع} = \frac{\bar{e}}{ع} = \frac{1}{2 |ع|} \text{ .}‬$
