جدول (٦ - ١)

لذا نلاحظ من الجدول (٦ - ١) أن : $‫\frac{\text{جاس}}{\text{س}} \leftarrow ١‬$‫ عندما س ←.

نتيجة (١) :

‬$‫1) \text{ نهياً } \frac{\text{جاس}}{\text{جاس}} = ١ \quad (2) \text{ نهياً } \frac{\text{جاس}}{\text{جاس}} = ١‬$‫

‬$‫3) \text{ نهياً } \frac{\text{ظاس}}{\text{س}} = ١ \quad (4) \text{ نهياً } \frac{\text{جاس}}{\text{س}} = ١‬$‫

مثال (٦ - ٢)

أوجد أ) نهياً ‬$‫\frac{\text{جاس}}{\text{س}} = ٦‬$‫

ب) نهياً س ظناس

الحل :

أ) بفرض أن : ص = ٥ س ← س = ‬$‫\frac{\text{ص}}{٥}‬$‫ ، أي أن : عندما س ← ، فإن ص ←.

‬$‫\therefore \text{ نهياً } \frac{\text{جاس}}{\text{س}} = \frac{\text{جاس}}{\text{س}} = \frac{\text{جاس}}{\text{ص}} = \frac{\text{جاس}}{\text{ص}} = \frac{٥}{٦} \times ٦ = \frac{٥}{٦}‬$‫

‬$‫\text{ب) نهياً س ظناس} = \text{نهياً } \frac{\text{ب}}{\text{ص}} = ١‬$‫

تدريب (٦ - ١)

أوجد نهاية الدوال التالية وفق الشروط المرافقة لكل منها :

‬$‫1) \text{ د(س)} = \frac{\text{جاس}}{\text{ب س}} \rightarrow \text{س} \leftarrow \text{س}‬$‫

‬$‫2) \text{ د(س)} = \text{س قنا س} \rightarrow \text{س} \leftarrow \text{س}‬$‫

‬$‫3) \text{ د(س)} = \text{س جاس} \frac{1}{\text{س}} \rightarrow \text{س} \leftarrow \infty‬$‫

‬$‫4) \text{ د(س)} = \frac{1 - \text{جنا س}}{\text{س}} \rightarrow \text{س} \leftarrow \text{س}‬$
