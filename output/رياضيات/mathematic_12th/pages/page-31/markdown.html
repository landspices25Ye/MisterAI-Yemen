الأعداد المركبة

مثال (١ - ١٨)

أوجد السعة لكل من الأعداد التالية :

أ) ع₁ = (جتا ه - ت جا ه) . ب) ع₂ = جا ه - ت جتا ه . ج) ع₃ = - جا ه - ت جتا ه .

الحل :

أ) ع₁ = جتا ه - ت جا ه = جتا ( - ه ) + ت جا ( - ه ) = جتا ( π ٢ - ه ) + ت جا ( π ٢ - ه ) .

∴ سعة ( ع₁ ) = π ٢ - ه = - ه .

ب) ع₂ = جا ه + ت جتا ه نحوها إلى الصورة القطبية العامة للعدد المركب

= جتا ( π/2 - ه ) + ت جا ( π/2 - ه ) ، [ جا ه = جتا ( π/2 - ه ) ؛ جتا ه = جا ( π/2 - ه ) ]

∴ سعة ( ع₂ ) = π/2 - ه .

ج) ع₃ = - جا ه - ت جتا ه = جتا ( π ٣/2 - ه ) + ت جا ( π ٣/2 - ه )

∴ سعة ( ع₃ ) = π ٣/2 - ه .

تدريب (١ - ٧)

أوجد السعة لكل من :

ع₁ = جا ه - ت جتا ه ، ع₂ = - جا ه + ت جتا ه .
