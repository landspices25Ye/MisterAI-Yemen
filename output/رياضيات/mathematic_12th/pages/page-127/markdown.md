القطوع الخروطة

مثال (٤ - ١٨)

بين نوع المنحنى الذي تمثله المعادلات التالية :

أ) س² + ٣ ص² + ٨ س + ١٢ ص + ٥ = ٠
ب) ٣ س² + ٢ ص² + ٦ س + ٤ ص + ١ = ٠
ج) ٣ س² + ٢ ص² + ٦ س + ٤ ص - ١ = ٠
د) ٢ س² + ٤ س - ص + ٣ = ٠

# **الحل :**

أ) بمقارنة المعادلة المعطاة مع المعادلة : س² + ج ص² + ه س + ح ص + ط = ٠ ، نجد أن ١ ، ج لهما الإشارة نفسها . وبالتالي فالمعادلة تمثل إما قطعاً ناقصاً ، أو دائرة ، أو نقطة ، أو مجموعة خالية . ولعرفة ذلك نكتب المعادلة بالصورة : (س² + ٨ س) + ٣ (ص² + ٤ ص) = ٥
وبإكمال المربع تأخذ المعادلة الأخيرة الصورة :

$‫(س² + ٨ س + ١٦) + ٣ (ص² + ٤ ص + ٤) = ٥ - ١٦ + ١٢‬$‫

‬$‫\Leftarrow (س + ٤) + ٣ (ص + ٢) = ٢٣‬$‫

‬$‫\Leftarrow \frac{(س + ٤) + ٢}{(س + ٢) + ٢٣} = ١‬$‫

بوضع س = س + ٤ ، ص = ص + ٢ ، أي نسحب المحاور للنقطة (٤ - ٢) فتأخذ

المعادلة الصورة : ‬$‫\frac{س^2}{٢٣} + \frac{ص^2}{٢٣} = ١‬$‫ وهذه معادلة قطع ناقص .

ب) المعادلة المعطاة تمثل قطعاً زائداً أو مستقيمين متقاطعين لأن ١ ، ج لهما إشارتان مختلفتان . لمعرفة ذلك يمكن كتابة المعادلة بالشكل :

‬$‫(ص + ٤ ص) - ٣ (س - ٢ س) = ١ -‬$‫

‬$‫\Leftarrow (ص + ٤ ص + ٤) - ٣ (س - ٢ س + ١) = ١ - ٤ - ٣‬$‫

‬$‫\Leftarrow (ص + ٢) - ٣ (س - ١) = ٠‬$‫

‬$‫\Leftarrow (ص + ٢) = ٣ (س - ١)‬$‫

‬$‫\Leftarrow ص + ٢ = \sqrt[٣]{٣} (س - ١) \Leftarrow ص = \sqrt[٣]{٣} س حيث ص = ص + ٢ ، س = س - ١‬$

∴ المعادلة تمثل مستقيمين متقاطعين .
