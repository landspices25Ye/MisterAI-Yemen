الحل :

أ) ∴ الصورة القطبية للعدد ع = م (جتا ه + ت جا ه) .

$‫\sqrt{2} = \sqrt{2(1) + 2(1)} = |ت + 1| = م‬$‫

‬$‫\therefore \text{جتا ه} = \frac{1}{\sqrt{2}} = \frac{1}{\sqrt{2}} \text{ ، جا ه} = \frac{1}{\sqrt{2}} = \frac{1}{\sqrt{2}} .‬$‫

∴ جتا ه < ٠ ، جا ه < ٠ ، ه تقع في الربع الأول .

‬$‫\therefore \frac{\pi}{4} = ه .‬$‫

الصورة القطبية للعدد (١ + ت) = 2√ (جتا 1/4 + ت جا 1/4) أو 1 + ت = [ 2√ ، 1/4 ] .

ب) ∴ ع = ١ - ص = ١ - ص = ٠ .

‬$‫\therefore \sqrt{2} = \sqrt{2 + 1} = \sqrt{2 + 2} = 1‬$‫

‬$‫\text{جتا ه} = \frac{1}{\sqrt{2}} = \frac{1}{\sqrt{2}} = 1 - \text{،}‬$‫

‬$‫\text{جا ه} = \frac{1}{\sqrt{2}} = \frac{1}{\sqrt{2}} = 0‬$‫

ه تنطبق على محور السينات السالب ، ∴ ه = π

‬$‫\sqrt{2} = (\text{جتا } \pi + \text{ت جا } \pi) = [ \pi \text{ ، } 1 ]‬$‫

وباعتبار (س ، ص) = (٠ ، ١ - ) ، ∴ ه = π
