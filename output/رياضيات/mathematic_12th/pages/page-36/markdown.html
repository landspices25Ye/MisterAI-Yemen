$‫\sqrt{27} \text{ ت} = \sqrt{27} \text{ (جتا } \frac{\pi}{2} + \text{ت جا } \frac{\pi}{2} \text{)} = [\frac{\pi}{2}, \sqrt{27}]‬$‫

‬$‫\therefore \frac{\sqrt{27}}{\sqrt{1+2}} = \frac{[\frac{\pi}{2}, \sqrt{27}]}{[\frac{\pi}{4}, \sqrt{27}]} = \frac{\sqrt{27}}{\sqrt{1+2}} = [\frac{\pi}{4}, 1] =‬$‫

‬$‫[\frac{\pi}{4}, 1] =‬$‫

‬$‫[\frac{\sqrt{27}}{\sqrt{1+2}} = [\frac{\pi}{4}, 1] = [\frac{\pi}{4}, 1] = [\frac{\pi}{4}, 1] = [\pi, 1] = [\pi, 1]‬$‫
