الوحدة السادسة

مثال (٦ - ١٩)

أوجد مشتقة الدالتين التاليتين :

أ) ص = $‫\frac{m}{m+1}‬$ ب) ص = $‫m^2‬$

الحل :

أ) باستخدام خواص اللوغاريتمات لإيجاد مشتقة الدالة المعطاة نجد أن :

ص = $‫\frac{m}{m+1}‬$ $‫\Leftarrow‬$ لو ص = لو $‫\frac{m}{m+1}‬$ = لو س - لو (س + ١)

$‫\therefore \frac{1}{m} \times \text{ص} = \frac{1}{m} - \frac{1}{m+1} \Leftarrow \text{ص} = \text{ص} \left( \frac{1}{m} - \frac{1}{m+1} \right)‬$‫

وبالتعويض عن ص = ‬$\frac{m}{m+1}$‫ نجد أن : ص = ‬$\frac{m}{m+1}$‫ ( ‬$\frac{1}{m} - \frac{1}{m+1}$‫ )

‬$‫= \frac{1}{(m+1)} - \frac{1}{(m+1)^2} = \frac{1}{(m+1)^2} = \frac{1}{(m+1)^3}‬$‫

ب) لكون الأساس والأس متغيرات في الدالة ص = ‬$m^2$‫ لا نستطيع تطبيق قواعد الاشتقاق ولذلك نقوم

بأخذ لوغاريتم طرفي المعادلة : لو ص = لو س = س لو س ،

وبتطبيق قاعدة الاشتقاق للدالة اللوغاريتمية نجد أن :

( لو ص ) = ( س لو س )

‬$‫\Leftarrow \frac{1}{m} \times \text{ص} = \text{ص} \times \frac{1}{m} + 1 \times \text{لو س} = 1 + \text{لو س}‬$‫

‬$‫\Leftarrow \text{ص} = \text{ص} ( 1 + \text{لو س}) = \text{ص} ( 1 + \text{لو س})‬$‫

ثانياً : مشتقة الدالة الأسية :

لإيجاد مشتقة الدالة الأسية ص = ‬$l^2$‫ ، المعرفة من ح ‬$\leftarrow$‫ ح + ‬$\rightarrow$‫ ح ‬$\leftarrow$‫ ح + ‬$\rightarrow$‫ ح ‬$\leftarrow$‫ / ‬$\leftarrow$‫ } يمكن كتابة

الدالة بالصورة س = لو ص وباشتقاق طرفي المعادلة بالنسبة لـ س نحصل على :

‬$‫1 = \frac{1}{m} \times \frac{5}{m} \times \text{لو ه} \Leftarrow \frac{5}{m} \times \frac{1}{m} = \text{ص} \times \frac{1}{\text{لو ه}} = \text{ص لو ه}‬$
