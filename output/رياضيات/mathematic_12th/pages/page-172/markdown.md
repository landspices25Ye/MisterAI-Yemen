## تمارين ومسائل (١-٤)

[١] أوجد مشتقة كل من الدوال الآتية :

أ)  $‫س^2 ص = ٤ س + ١‬$

ب)  $‫\frac{1}{س} + \frac{1}{ص} = ٢‬$

ج)  $‫س^2 + \sqrt{ص ص} = ٧‬$

د)  $‫س^2 + ص^2 = ٢ س ص‬$

هـ)  $‫س^2 + \sqrt{ص ص} = ٣‬$

[٢] أوجد  $‫\frac{ص^4}{ص^3}‬$  عند النقاط الموضحة أمام كل منها :

أ)  $‫س^2 + \frac{ص^2 ص}{\pi} = ٣‬$  ، عند النقطة ( ١ ، ٣ )

ب)  $‫س^2 + ٤ ص^2 + ٧ = ٢‬$  ، عند النقطة ( ١ ، ١ )

[٣] أوجد قياس الزاوية التي يصنعها المماس لمنحنى الدالة  $‫س^2 + ص^2 - ٢ س + ٦ ص + ٢ = ٠‬$

عند النقطة ( ١ - ٣ ) مع محور السينات في الاتجاه الموجب .
