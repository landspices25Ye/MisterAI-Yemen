# جدول (٣) هرمونات الفص الأمامي للغدة النخامية

|  الهرمون | العضو المتأثر بالهرمون | الوظيفة  |
| --- | --- | --- |
|  ١- هرمون النمو (STH). | عظام الجسم والعضلات في الأجهزة. | - تنظيم نمو الجسم خلال مراحل العمر المختلفة ببناء العظام، ونموها. - بناء البروتينات، وتنظيم عملية الأيض.  |
|  ٢- الهرمون المنشط للحويصلات في المناسل (FSH). | - مبيض الأنثى. | - نمو الحويصلات في المبيض، وتحويلها إلى حويصلات جراف.  |
|  ٣- الهرمون المنشط للجسم الأصفر (LH). | - خصبة الذكر | - يساعد على تكوين الانبيجبات المنوية في الخصبة، وتكوين الحيوانات المنوية.  |
|  ٤- الهرمون المنشط للخصبة (ICSH). | - مبيض الأنثى. | - يعمل على نضج البويضة وخروجها من حويصلة جراف. - يحفز الخلايا البينية على تكوين الجسم الأصفر.  |
|  ٥- الهرمون المفرز للحليب (البرولاكين). | - خصبة الذكر. | - يحفز الخلايا البينية على إنتاج الهرمونات الجنسية. «التستوسيترون».  |
|  ٦- الهرمون الحافز للغدة الدرقية (TSH). | - الغدة اللبينية في الثديين. | - يعمل على نمو الغدة اللبينية، وإفراز الحليب منها.  |
|  ٧- الهرمون الحافز لقشرة الغدة الكظرية (ACTH). | - الغدة الدرقية. | - يحفز الغدة على إنتاج هرمون (الثيروكسين). - يعمل على تجميع اليود في نسيج الدرقية.  |
|  ٨- الهرمون المنشط لخلايا الميلانين (MSH). | - قشرة الكظرية. | - يحفز الغدة على إفراز هرموناتها.  |
|   | - الجلد. | - يعمل على تحفيز إفراز صبغ الميلانين التي تصبغ الجلد باللون الأسمر.  |
