### حساب الأطوال الموجية لطيف ذرة الهيدروجين :

وفقاً لفرضية بوهر الثالثة التي تنص على أنه إذا كانت طاقة الإلكترون الابتدائية هي ( طأ ) وانتقل إلى حالة نهائية طاقتها ( طم ) أصغر من طاقة الحالة ( طأ ) ، فإن ذرة الهيدروجين تبعث بضوء طاقته ( hf ) تحسب من العلاقة التالية :

$$hf = طأ - طم$$

$$\text{أو } f = \frac{طأ}{h} - \frac{طم}{h}$$

ولكن $f = \frac{\frac{\text{ع}}{\text{ص}}}{\frac{\text{ع}}{\text{ص}}}$ ( حيث ( ع ) سرعة الضوء المنبعث و ( $\lambda$ ) طول موجته و ( f ) تردده ،

$$\text{وأن } \text{طأ} = \frac{\text{طأ}}{\text{نأ}} \text{ ، طم} = \frac{\text{طأ}}{\text{نأ}} \text{ ( من العلاقة ١٣ ) .}$$

وبتعويض هذه القيم في العلاقة ( أعلاه ) نجد أن :

$$\frac{\text{ع}}{\lambda} = \frac{\text{طأ}}{\text{نأ}} - \frac{\text{طأ}}{\text{نأ}} = \frac{\text{طأ}}{\text{نأ}} \left( \frac{1}{\text{نأ}} - \frac{1}{\text{نأ}} \right)$$

$$\text{أو } \frac{1}{\lambda} = \frac{\text{طأ}}{\text{ع}} \left( \frac{1}{\text{نأ}} - \frac{1}{\text{نأ}} \right) \text{ وحيث أن طأ} = - ١٣,٦ \text{ ( ف. } \text{أ. } \text{ف)}$$

$$\therefore \frac{1}{\lambda} = \frac{١٣,٦}{\text{ع}} \left( \frac{1}{\text{نأ}} - \frac{1}{\text{نأ}} \right)$$
