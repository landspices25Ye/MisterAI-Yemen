الاحتمالات

$$\therefore \text{حا}(t) + \text{حا}(n) - \text{حا}(t) = \text{حا}(t) \text{حا}(n) + \text{حا}(n)$$
$$= \text{حا}(t) [1 - \text{حا}(n)] + \text{حا}(n)$$
$$= \text{حا}(t) - \text{حا}(t) \text{حا}(n) + \text{حا}(n)$$

$$\therefore - \text{حا}(t) = - \text{حا}(t) \text{حا}(n) \Leftarrow \text{حا}(t) = \text{حا}(t) \text{حا}(n)$$
$$\therefore t, n \text{ حادثتان مستقلتان.}$$

# **مثال (٣ - ١٢)**

إذا كانت $t \supset n$ وكان $\text{حا}(t) = \frac{1}{4}$، $\text{حا}(n) = \frac{1}{3}$، فهل $t, n$ حادثتان مستقلتان؟

# **الحل:**

نبحث في تحقق الشرط: $\text{حا}(t) = \text{حا}(t) \text{حا}(n)$.

أولاً: نوجد قيمة الطرف الأيمن من الشرط كما يلي:

$$\therefore t \supset n \Leftarrow t = \text{حا}(t) \Leftarrow \text{حا}(t) = \text{حا}(t) = \frac{1}{4} \dots\dots\dots (1)$$

ثانياً: نوجد قيمة الطرف الأيسر من الشرط كما يلي:

$$\text{حا}(t) \text{حا}(n) = \frac{1}{4} \times \frac{1}{3} = \frac{1}{12} \dots\dots\dots (2)$$

ثالثاً: نقارن بين (١)، (٢) فنجد أن:

$$\frac{1}{4} \neq \frac{1}{12} \text{ أي أن: } \text{حا}(t) \neq \text{حا}(t) \text{حا}(n)$$

$\therefore$ لم يتحقق شرط الاستقلالية للحادثتين.

$\therefore t, n$ حادثتان غير مستقلتين.

# **تمارين ومسائل (٣ - ٣)**

[١] إذا كان: $\text{حا}(t) \mid n = 0,4$، $\text{حا}(n) = 0,8$. فأوجد $\text{حا}(t) n$

[٢] إذا كانت $t, n$ حادثتين مستقلتين، $\text{حا}(t) = \frac{1}{2}$، $\text{حا}(t) \cap n = \frac{2}{3}$

فأوجد: (١) $\text{حا}(n)$ (٢) $\text{حا}(t) \mid n$ (٣) $\text{حا}(n) \mid t$
