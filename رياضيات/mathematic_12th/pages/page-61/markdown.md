# نتيجة ( ١ ) :

$$\text{م}' = \text{م}' + \text{م}' - 1$$

البرهان :

$$\therefore \text{الطرف الأيمن} = \text{م}' + \text{م}' - 1 = \frac{\text{م}'}{\text{م}' - \text{م}' + \text{م}' - 1}$$

$$= \frac{\text{م}'}{\text{م}' - \text{م}' + \text{م}' - 1} + \frac{\text{م}'}{\text{م}' - \text{م}' + \text{م}' - 1}$$

$$= \frac{\text{م}' + \text{م}' - 1}{\text{م}' - \text{م}' + \text{م}' - 1} = \frac{\text{م}' + \text{م}' - 1}{\text{م}' - \text{م}' + \text{م}' - 1} = \frac{\text{م}' + \text{م}' - 1}{\text{م}' - \text{م}' + \text{م}' - 1}$$

$$= \frac{\text{م}' + \text{م}' - 1}{\text{م}' - \text{م}' + \text{م}' - 1} = \frac{\text{م}' + \text{م}' - 1}{\text{م}' - \text{م}' + \text{م}' - 1}$$

$$= \text{م}' + \text{م}' = \text{الطرف الأيسر}$$

$$\therefore \text{م}' + \text{م}' - 1 = \text{م}' + \text{م}'$$

وتسمى هذه العلاقة « علاقة الكرخي » .

خواص التوافق :

$$1 - \text{م}' = \text{م}' - \text{م}'$$

$$2 - \text{إذا كان م}' = \text{م}' + \text{م}' - 1$$ ، فإما م' = م' ، أو م' = م' + م' = م' .
