الوحدة السابعة

$$\Delta m = \frac{1-\mu}{\rho} = \frac{4}{\rho}$$

$$\rho \frac{4}{\rho} = \frac{1-\mu}{\rho} + 1 = \frac{4}{\rho} \text{ نختار}$$

$$\rho \left( \frac{4}{\rho} \right) = \rho \left( \frac{4}{\rho} \right) = 1 + \rho \frac{4}{\rho}$$

$$\rho \frac{4}{\rho} \times \rho \left( \frac{4}{\rho} \right) = \rho \left( \frac{4}{\rho} \right) \rho \left( \frac{4}{\rho} \right) = \rho \left( \frac{4}{\rho} \right) \rho \left( \frac{4}{\rho} \right)$$

$$= \left[ \frac{4}{\rho} \right] \left[ \frac{4}{\rho} \right] = \rho \left( \frac{4}{\rho} \right) = \rho \left( \frac{4}{\rho} \right)$$

$$= \left[ \frac{4}{\rho} \right] \left[ \frac{4}{\rho} \right] \times \left[ \frac{4}{\rho} \right] = \left[ \frac{4}{\rho} \right] \times \left[ \frac{4}{\rho} \right]$$

$$= \frac{4}{\rho} \left( \frac{4}{\rho} \right) = \frac{12}{\rho} = \frac{8}{\rho}$$

واستناداً إلى الصيغة (١-٧) نجد أن:

$$\text{سط} = \text{نهي} \quad \text{نهي} \quad 12 = \frac{8+12}{\rho} \text{ وحدة مربعة.}$$
