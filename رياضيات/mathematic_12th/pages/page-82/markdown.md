$$\text{حا (د) = حا (ص، ك) + حا (ك، ص) + حا (ك، ك)}$$

لاحظ من حل المثال السابق (٣ - ٣) فقرة (٢) أن كل نقطة من نقاط فضاء العينة « ع » لها الاحتمال

$$\text{نفسه ويساوي } \frac{1}{\sqrt{(ع)}} = \frac{1}{4}$$

الفضاءات التي تكون فيه فرصة وقوع الحوادث متساوية تسمى الفضاءات ذات الاحتمالات المتساوية ، وتُسمى دالة الاحتمال ( حا ) المعرَّفة على مثل هذه الفضاءات دالة الاحتمال المنتظمة فمثلاً : عند إلقاء حجر نرد وملاحظة العدد الظاهر على الوجه العلوي له عند استقراره على الأرض ، فإن : $$\text{ع} = \{ 1, 2, 3, 4, 5, 6, 7 \}$$ ، ونجد أن :

$$\text{حا (أ) = حا (ب) = حا (ج) = حا (د) = حا (هـ) = حا (و) = حا (ز) = حا (ح) = 1/6}$$

وبهذا نكون قد خصصنا في هذه التجربة لكل نقطة من نقاط فضاء العينة « ع » الاحتمال نفسه = $$\frac{1}{\sqrt{(ع)}} = \frac{1}{6}$$ وهذا يعني أن ( حا ) دالة احتمال منتظمة ؛ إلا أن بناء النموذج الاحتمالي لا يفترض تساوي قيم ( أ ) دائماً في كل التجارب العشوائية . وإذا افترضنا الآن أن ( أ ) حادثة تتضمن ( أ ) حادثة ابتدائية ( أولية ) فإن :

$$\text{حا (أ) = } \underbrace{\frac{1}{\sqrt{(ع)}} + \frac{1}{\sqrt{(ع)}} + \dots + \frac{1}{\sqrt{(ع)}}}_{\text{حا (أ) مرة (حيث احتمال كل نقطة (حادثة ابتدائية) = } \frac{1}{\sqrt{(ع)}})}$$

$$\text{حا (أ) = } \underbrace{\frac{1}{\sqrt{(ع)}} + \dots + \frac{1}{\sqrt{(ع)}} + \frac{1}{\sqrt{(ع)}}}_{\text{حا (أ) مرة (حيث احتمال كل نقطة (حادثة ابتدائية) = } \frac{1}{\sqrt{(ع)}})}$$
