الهندسة الفضائية

المطلوب : حساب قياس الزاوية الزوجية بين المستويين ( ا ب و ) ، ( ج ب و )

الحل : نصف ب و في النقطة و

$$\therefore | ب ج | = | و ج | \quad (معطى)$$

$$\therefore \overline{ج و ل و ب} \quad (خواص المثلث المتساوي الساقين)$$

$$\therefore | ج | \quad (معطى) \quad (ب ج و)$$

$$\therefore \overline{ج و ل و ب} \quad (استنتاجاً)$$

$$\therefore | و | \quad (مبرهنة الأعمدة الثلاثة)$$

$$\therefore \text{المستوى } ( ا ب و ) \cap \text{المستوى } ( ج ب و ) = \overline{ب و}$$

$$\therefore \text{ (ا و ج) هي الزاوية الخطية للزاوية الزوجية بين المستويين (ا ب و) ، (ج ب و)}$$

لنرمز لها بالرمز هـ .

$$\therefore \text{ ب ج و } \Delta \text{ متساوي الساقين وقائم في ج } \quad \therefore \text{ و } ( \text{ب ب } ) = \text{ و } ( \text{و ب } ) = ٤٥$$

$$\text{جا } ( \text{و ب } ) = \frac{| و ج |}{| و ج |} \Leftarrow \frac{1}{\sqrt{7}} = \frac{| و ج |}{10}$$

$$\therefore | و ج | = \frac{10}{\sqrt{7}} \Leftarrow | و ج | = 5 \sqrt{7} \text{ سم}$$

$$\therefore | ج | = 5 \sqrt{7} \text{ سم}$$

$$\therefore \text{ظاه} = \frac{| ج |}{| و ج |}$$

$$\therefore \text{ظاه} = \frac{5 \sqrt{7}}{5 \sqrt{7}} =$$

$$\therefore \text{ظاه} = \sqrt{7}$$

$$\therefore \text{هـ} = 6$$
