مثال (٧ - ١٤)

أوجد قيم ج التي تحقق مبرهنة القيمة المتوسطة للتكامل الآتي :

$$\left. \begin{array}{l} ٣ \\ ٢- \end{array} \right\} (٣ \text{ س } ٢ + ٢ \text{ س}) \text{ وس} .$$

الحل :

$$\left. \begin{array}{l} ٣ \\ ٢- \end{array} \right\} (٣ \text{ س } ٢ + ٢ \text{ س}) \text{ وس} = (٣ \text{ ج } ٢ + ٢ \text{ ج}) (٣ - ٢ - ٢) .$$

$$(٣ \text{ س } ٢ + ٢ \text{ ج}) = \left. \begin{array}{l} ٣ \\ ٢- \end{array} \right\} (٣ \text{ س } ٢ + ٢ \text{ ج}) .$$

$$٢٧ + ٩ - (٨ - ٤) = ٥ (٣ \text{ ج } ٢ + ٢ \text{ ج}) .$$

$$٥ (٣ \text{ ج } ٢ + ٢ \text{ ج}) = ٤٠ \Leftarrow ٣ \text{ ج } ٢ + ٢ \text{ ج} = ٨ .$$

$$٣ \text{ ج } ٢ + ٢ \text{ ج} = ٨ = ٠ \Leftarrow (٣ \text{ ج } ٢ - ٤) (٣ \text{ ج } ٢ + ٢) = ٠ .$$

$$\therefore \text{ أما } \text{ ج} = \frac{4}{3} \text{ ، أو } \text{ ج} = -٢ .$$

$$\text{نلاحظ أن : } \frac{4}{3} \text{ و } [٣, ٢] \text{ و } [٣, ٢] \text{ و } [٣, ٢] \text{ و } [٣, ٢] .$$

$$\text{إذن ل ج قيمتان تحققان المبرهنة هما : } -٢, \frac{4}{3} .$$
