# **مثال (٢ - ٢٣)**

اكتب الحد السابع في مفكوك (٢ س - ص)⁹.

**الحل:** باستخدام الحد العام $ح ر + ١ = ٥$ و $ح ر + ٢ = ٧$ يكون:

$$ح ر = ١ + ٦ = ٩ \text{ و } ٦ (٢ س) (٣ - ص) = \frac{٩}{٣} (٨ س) (٣ - ص) = \frac{٩ \times ٨ \times ٩}{٦} \times ٨ \times ٢ = ٨ \times ٨ \text{ ص} = ٦٧٢ \text{ ص} = ٦.$$

# **مثال (٢ - ٢٤)**

أوجد الحد الثامن في مفكوك (٢ س + ١ س)¹⁰.

**الحل:**

$$ح ٨ = ١ + ٧ = ١٠ \text{ و } ٧ (٢ س) (٣ - ص) = ٧ \frac{١}{٣} (٢ س) (٣ - ص) = ٧ \frac{١}{٣} (٢ س) = ٧ \frac{١}{٣} (٢ س) = ٩٦٠ \text{ ص}.$$

# **مثال (٢ - ٢٥)**

أوجد معامل س¹⁰ في مفكوك (٢ س - ٣)¹⁴.

**الحل:**

بما أن رتبة الحد الذي يحتوي س¹⁰ غير معلومة فإننا نفرض أن الحد الذي يحتوي س¹⁰ هو ح ر + ١ وعليه يكون:

$$ح ر + ١ = ١٤ \text{ و } ١٤ (٢ س) (٣ - ص)$$

ولكي نحصل على الحد الذي يحتوي س¹⁰ نضع (س)¹⁴ - ص = (س)¹⁰

$$\therefore ١٤ - ص = ١٠ \text{ ، } ص = ١٤ - ١٠ = ٤$$

$$\therefore ح = ١٤ \text{ و } ١٤ (٢ س) (٣ - ص) = ١٤ \frac{١٤}{٤} \times ١٠٢ \times ١٠٢ \times ٤$$

$$= \frac{١٤ \times ١٣ \times ١٢ \times ١١ \times ١٠}{٤ \times ٣ \times ٢ \times ١} \times ١٠٢ \times ١٠٢ \times ٤$$

∴ معامل س¹⁰ هو ١٠٢ × ٨١٠٨١.
