$$\therefore 1 = \{ (و، ب، ب)، (ب، و، ب)، (ب، ب، و) \} \Leftarrow \text{و (1)} = 3$$

ب) نفرض أن : ب هي حادثة أن يكون الطفل الأكبر للأسرة المختارة ولداً .

$$\therefore \text{ب} = \{ (و، و، و)، (و، و، ب)، (و، ب، و)، (و، ب، ب) \} \Leftarrow \text{و (ب)} = 4$$

$$\therefore \text{حا (ب)} = \frac{\text{و (ب)}}{\text{و (ع)}}$$
