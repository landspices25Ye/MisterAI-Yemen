$$( 1,1, 1,2, 1,3, 1,5 ) \text{ أمبير .}$$

د- عند مرور تيار متردد في ملف حثي فإن الملف يولد ( ق. د. ك ) تأثيرية اتجاهها يكون في اتجاه ... ( القوة المولدة لها - معاكس لاتجاه القوة المولدة لها - كل ما ذكر صحيح )

ز- يمكن حساب المفاعلة السعوية لمكثف متصل بمصدر تيار متردد من العلاقة
